# Budget Calculator

Небольшой учебный проект на JavaScript для расчёта личного бюджета.

## Описание
Пользователь вводит доход и расходы и получает информацию
об остатке средств с визуальной индикацией состояния.

## Функциональность
- ввод дохода и расходов
- валидация пользовательского ввода
- расчёт остатка бюджета
- статусы: остаток / ноль / перерасход
- цветовая индикация результата

## Используемые технологии
- HTML
- CSS
- JavaScript (DOM, events, classList)
